# 🌌 VivaVox: Neural Interview Intelligence

VivaVox is a state-of-the-art, **independent desktop application** designed to revolutionize the interview experience through on-device AI and immersive 3D technology.

## 🚀 Key Features

- **"No Compromise" 3D Design:** An immersive 3D neural environment that reacts to your voice in real-time.
- **Independent On-Device AI:** Powered by local **Gemma** models. Your data never leaves your machine—ensuring total privacy and zero latency.
- **HUD Interview Interface:** A futuristic Heads-Up Display (HUD) that provides real-time analytics (eye tracking, tone, confidence) during the interview.
- **Neural Reporting:** Automatically generates high-quality PDF reports and sends them directly to students via secure SMTP.
- **Uncrashable Shell:** Built with **Tauri** and **Rust** for maximum stability and a tiny footprint.

## 🛠️ Tech Stack

- **Frontend:** Next.js 14, React Three Fiber (Three.js), Tailwind CSS, Framer Motion.
- **Desktop Core:** Tauri (Rust).
- **AI Models:** Local Gemma (LLM) & Whisper (Speech-to-Text).
- **Reporting:** Local PDF engine & Rust-based SMTP service.

## 📦 Getting Started

### Prerequisites

1.  **Rust:** Install from [rust-lang.org](https://www.rust-lang.org/tools/install).
2.  **Node.js:** Ensure you have Node.js installed.
3.  **Tauri CLI:** `npm install -g @tauri-apps/cli`.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/[your-username]/vivavox-desktop.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Launch in Development Mode:**
    ```bash
    npm run tauri dev
    ```
